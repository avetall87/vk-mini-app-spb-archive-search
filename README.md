# Создать VK Mini App [![npm][npm]][npm-url] [![deps][deps]][deps-url]

## Приложение для поиска архивных документов по сайту - https://medal.spbarchives.ru/

### Создать приложение VK Mini App с деплоем на github pages - gh-pages deploy

`npx @vkontakte/create-vk-mini-app <app-directory-name>`

## Работа с приложением

Перейти в директорию и запустить:
`yarn start` || `npm start` — запуск сервера разработки с hot reload по адресу `localhost:10888`.

`yarn run build` || `npm run build` — создание сборки для промышленного контура
